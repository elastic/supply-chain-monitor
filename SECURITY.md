# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability in this project, please
report it responsibly through Elastic’s security disclosure process:

[https://www.elastic.co/community/security](https://www.elastic.co/community/security)

Please do **not** file public issues for undisclosed security problems.

## Scope

Reports should relate to this repository’s code and documentation. Issues in
third-party services (PyPI, npm, Cursor, Slack, and so on) should be reported to
those vendors directly.
